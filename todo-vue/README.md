# Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).

---

## Levantar el Frontend

Sigue estos pasos para ejecutar la aplicación de Vue localmente:

1. **Instalar dependencias**
   ```bash
   cd todo-vue
   npm install
   ```

2. **Arrancar el servidor de desarrollo**
   ```bash
   npm run dev
   ```

   El proyecto estará disponible en `http://localhost:5173` (o el puerto que muestre la consola).

3. **Construir para producción** (opcional)
   ```bash
   npm run build
   ```

   Los archivos generados se colocarán en `dist/`.

4. **Servir la compilación** (opcional)
   ```bash
   npm run preview
   ```

   Esto levanta un servidor estático para previsualizar la versión de producción.
